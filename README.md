# 🏠 House Sales Analysis – King County, USA

## 📌 Project Overview

This project is part of the IBM Data Analysis course. It focuses on analyzing house sales data from King County, USA to identify trends and factors affecting house prices.

The objective is to perform data cleaning, exploratory data analysis (EDA), and build a regression model to predict house prices.

---

## 📊 Dataset

* Source: King County House Sales Dataset
* Number of records: ~21,000
* Features include:

  * Price
  * Bedrooms and Bathrooms
  * Square Footage (Living Area & Lot)
  * Floors
  * Waterfront View
  * Condition and Grade
  * Year Built and Renovated

## 🔍 Project Tasks

### Data Cleaning

* Handled missing values
* Checked and corrected data types
* Removed duplicates

### Exploratory Data Analysis (EDA)

* Analyzed price distribution
* Studied relationships between features and price
* Visualized data using plots and graphs

### Feature Engineering

* Selected important features
* Prepared data for modeling

### Model Building

* Built regression models
* Evaluated model performance using R² score

---

## 📈 Key Insights

* Larger houses (higher square footage) generally have higher prices
* Waterfront properties are significantly more expensive
* House grade and condition strongly influence price
* Number of bedrooms alone is not a strong predictor
